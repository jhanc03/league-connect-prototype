### League Connect Social App Prototype

This is a small proof-of-concept prototype created as part of my final assessment for the Cross-Platform Development module of my university degree. Achieved a 59% overall for the module.  
The application was devised as a social networking app for players of the MMO League of Legends. The app would incorporate several features, such as:
  * Messaging - send messages to other users
  * Game Invites - invite users to a party
  * Discover - find other players based on set criteria (rank, region, etc.)
